== Description ==

It is a widget plugin which shows the last comment in the sidebar.

[Plugin Homepage](http://rubensargsyan.com/wordpress-plugin-the-last-comment)

== Installation ==

1. Download the-last-commentr.zip, unzip it and upload the the-last-comment directory (including all files within) to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Go to the "Appearance"->"Widgets" panel and drag-and-drop the "The Last Comment" box into your sidebar, configure options and save.
